Mapa do Site

Guia de unificação para o ecossistema Independência Hub

1. Landing Page Principal (index.html) Ação Necessária

A porta de entrada do cliente e o motor de captura de leads.

Correções:
  • Substituir SEU_WEBHOOK_MAKE_URL pelo link gerado no Make.com.
  • Revisar o caminho do favicon: o código busca ./favicon.ico, mas o arquivo enviado é favicon.ico.
  • Descomentar as tags de Google Analytics e Pixel quando as contas forem criadas.
Ver Página →
2. Conversão e Vendas (planos.html / hotmart.html) Ação Necessária

Onde o usuário escolhe entre o PDF (Escalável) ou a Consultoria (High Ticket).

Correções:
  • Unificar o --accent: aqui está usando um marrom, enquanto a index usa dourado.
  • Substituir SEU_LINK_HOTMART pelos links de checkout reais.
  • Em hotmart.html, incluir o link do seu Calendly no campo de agendamento.
Ver Planos →
3. Expansão (parceiros.html / galo.html) Estrutura OK

Sistema de embaixadores e moderação de projetos da comunidade.

Observações Técnicas:
  • Certificar que a view do Airtable em parceiros.html está com a opção "Shared View" ativa.
  • Substituir SEUNUMERO no link do WhatsApp pelo formato 55149...
Ver Parceiros →
4. Autoridade (guia.html / habite-se.html / conceito.html) Ação Necessária

Conteúdo que tira dúvidas e gera comissões via afiliados.

Unificação Visual:
  • habite-se.html está muito simples. Aplicar as classes de CSS da index.html para manter o padrão "Premium".
  • Substituir as imagens de placeholder.com no Guia de Compras por fotos reais do seu catálogo.
  • Trocar todos os SEU_LINK_AFILIADO_AQUI.
Ver Guia →
5. Bastidores (ia.html / aplicacoes.html / ferramentas.html) Pronto

Exposição da tecnologia usada (Sweet Home 3D, D5, IA).

Dica: Estas páginas são ótimas para atrair outros profissionais via SEO. Adicione palavras-chave como "Arquitetura com IA" nos metadados.
